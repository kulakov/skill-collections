# Сохранение контекста

Проанализируй историю чата с AI и создай структурированное резюме для переноса в новую сессию.

## Формат вывода:

### КОНТЕКСТ ПРОЕКТА
- Название/тема проекта
- Основная цель
- Ключевые ограничения

### ПРИНЯТЫЕ РЕШЕНИЯ
Что было решено в ходе обсуждения:
1. ...
2. ...
3. ...

### ТЕКУЩИЙ СТАТУС
- Что уже сделано
- На чем остановились
- Что в процессе

### ОТКРЫТЫЕ ВОПРОСЫ
- Что осталось нерешенным
- Что требует дальнейшего обсуждения

### КЛЮЧЕВЫЕ АРТЕФАКТЫ
- Созданные файлы
- Написанный код
- Документы

### СЛЕДУЮЩИЕ ШАГИ
Что нужно сделать дальше:
1. ...
2. ...

### ПРОМПТ ДЛЯ ПРОДОЛЖЕНИЯ
Готовый текст для начала новой сессии:

```
Продолжаем работу над [проект].

Контекст:
- [ключевые решения]
- [текущий статус]

Нужно:
- [следующие шаги]
```

История чата:
$ARGUMENTS
