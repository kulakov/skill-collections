# Загружалка — создание персонажа через интервью

Глубокое психологическое интервью для создания персонажа LARP в сеттинге средневековой Японии (по мотивам фильмов Куросавы).

## Источник промпта

Полный промпт: `~/09-REFERENCE/my-promts/Игры/Загружалка - Dropbox Paper.md`

## Роли проводников

1. **Старый монах Тэнсю** — Путь Долга (честь, клятвы, верность)
2. **Слепая предсказательница Омицу** — Путь Голода (забота о близких, семья)
3. **Мастер чайной церемонии Соэки** — Путь Страха (амбиции, статус)

## Психологические модели

### Акцентуации по Леонгарду/Личко
- Гипертимный, Циклоидный, Эмотивный
- Тревожный, Застревающий, Педантичный
- Демонстративный, Возбудимый, Дистимический, Экзальтированный

### Архетипы по Пирсон
- Невинный, Сирота, Воин, Заботящийся
- Искатель, Любовник, Разрушитель, Творец
- Правитель, Маг, Мудрец, Шут

### Архетипические фигуры для взаимодействия
- Антагонист, Наставник, Спутник, Помощник
- Искуситель, Страж порога, Вестник, Тень
- Трикстер, Анима/Анимус

## Фазы исследования

### Фаза 1: Глубокий самоанализ
- Паттерны, когнитивные искажения, защиты
- После каждых 5 вопросов — предложение перейти дальше

### Фаза 2: Драматические развилки
- Путь света vs Путь тьмы
- Первый выбор ДО игры

### Фаза 3: Вызов истины
- Выбор из персонажей источников
- Безжалостная правда о персонаже
- 5+ узлов судьбы для связи с другими

## Финальный портрет

- 5 главных черт/конфликтов
- Акцентуации и архетипы
- Модус операнди
- Вероятная судьба (Путь Света / Путь Тьмы)
- Узлы судьбы с формулой конфликта
