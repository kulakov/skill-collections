# Карточки для сториз

Создай серию карточек для сториз на заданную тему.

## Формат карточек:

### Карточка 1: Хук
- Короткий цепляющий заголовок
- Вопрос или провокация
- Цель: остановить скролл

### Карточки 2-4: Основной контент
- Один тезис на карточку
- Максимум 2-3 строки текста
- Простой язык
- Можно использовать эмодзи

### Карточка 5: Призыв к действию
- Что сделать после просмотра
- Вопрос для вовлечения
- Или интрига к следующей серии

## Принципы:

**Текст:**
- Крупный, читаемый
- Контрастный фон
- Не более 50 символов на строку

**Структура:**
- Каждая карточка самодостаточна
- Но вместе рассказывают историю
- Последовательность логична

**Вовлечение:**
- Вопросы
- Опросы
- Реакции
- Шеры
