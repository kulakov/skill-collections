# LARP Director Box — полная документация

Создание полного пакета документации для проведения LARP другими мастерами.

## Источник промпта

Полный промпт: `/Users/lance/lance-claude/09-REFERENCE/my-promts/Игры/LARP Director Box.md`

## Структура документации

### 1. Cover Section
- Game title, version, date
- Authors and contributors
- License information

### 2. Game Announcement
- **Short version** (social media)
- **Full version** (convention submission)
- Hook, player count, duration, age restrictions
- Content warnings, dress code

### 3. Game Requirements

#### Time Requirements
- Preparation, setup, briefing
- Workshop, main game, debrief
- Cleanup, buffer time

#### Participants
- Min/max players, GM count
- Support staff, gender ratio
- Backup staff, rotation schedule

#### Space Requirements
- Room dimensions, light/sound control
- Power, ventilation, emergency exits
- Water access, storage

#### Equipment List
- Technical (primary/backup)
- Furniture (essential/optional)
- Props (permanent/consumable)
- Documentation (master/handouts)
- Emergency kit

### 4. Setup Instructions
- Layout diagrams with measurements
- Equipment placement maps
- Light/sound setup schemes
- Power cable routing

### 5. Game Flow
- Detailed timeline table
- Phase transitions
- Emergency phase skip protocols

### 6. Technical Support
- Sound management (playlist, effects, volumes)
- Lighting schemes by phase
- Emergency lighting plan

### 7. Troubleshooting Guide
- Technical failures
- Player conflicts
- Medical emergency
- Security issues

### 8. Appendices
- Setup/equipment checklists
- Safety briefing script
- Workshop scripts
- Character sheets templates
- Emergency contacts

### 9. Quality Control Checklist
- Pre-game verification
- During game monitoring
- Post-game documentation
