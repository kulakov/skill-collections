# Метод Алгоритмов для LARP

Этот скилл помогает создать ролевую игру через пошаговый процесс с использованием проверенных алгоритмов.

## Источник промпта

Полный промпт: `/Users/lance/lance-claude/09-REFERENCE/my-promts/Игры/Метод алгоритмов.md`

## Как работает

1. **Анализ источника вдохновения** — литература, история, мифология
2. **Определение типов действующих лиц** — социальные слои, касты, роли
3. **Определение типов взаимодействий** — торговля, война, дипломатия, магия
4. **Определение предметов конфликтов** — власть, ресурсы, запреты, религия
5. **Схематическая модель** — кто с кем конфликтует, за что борются
6. **Выбор алгоритма** — False Paths / Partisans / Old House / Taning / Intrigue
7. **Разработка структуры** — Движок, Правила, Сеттинг, Стартовое положение, Роли

## Принципы

- **Не выдавать всё скопом** — делать остановки
- **3 контрастных варианта** на каждом шаге
- **1 вопрос** после каждого шага
- **Подтверждение** перед переходом к следующему шагу

## Доступные алгоритмы

1. **Taning** — [описание из альманаха]
2. **Havskaya** — методы Хавской
3. **False Paths** — ложные следы
4. **Partisans** — партизанские сюжеты
5. **Old House** — старый дом
6. **Intrigue** — интриги
