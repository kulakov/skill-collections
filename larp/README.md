# Работа LARP-мастера

Скиллы для разработки и проведения живых ролевых игр: сценарий, сеттинг, персонажи, механики, документация, портреты.

**33 скиллов.** Положи папки/файлы в `~/.claude/skills/` и `~/.claude/commands/`.

## Состав

- **l-g-algo** — Создание LARP через алгоритмы
- **l-g-archivist** — Нежный Архивист — строитель живых миров из лора
- **l-g-box** — Упаковка LARP в коробку для проведения без автора
- **l-g-call** — Анализ транскриптов созвонов по LARP
- **l-g-canon** — Хранитель Канона — систематизатор лора в source of truth
- **l-g-catalog** — Каталог 365 механик: 185 метатехник LARP + 180 механик настольных игр (BGG)
- **l-g-cats** — CATS Framework для LARP — быстрое выстраивание ожиданий перед игрой
- **l-g-char** — Создание персонажа для LARP через психологическое интервью
- **l-g-check** — Проверка целостности LARP через призму Nordic подхода
- **l-g-check-mechanic** — Проверка игровой механики на последствия
- **l-g-create** — Помощник по разработке сценариев LARP

- **l-g-edit** — Редактирование прозы и персонажей для LARP/литературы
- **l-g-extract** — Извлечение идей из конспектов по геймдизайну
