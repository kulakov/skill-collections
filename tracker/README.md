# Работа трекера стартап-команд

Скиллы для сессий трекинга: подготовка, оркестрация, дебриф, методологии TOC/TRIZ, разбор созвонов в тезисы.

**13 скиллов.** Положи папки/файлы в `~/.claude/skills/` и `~/.claude/commands/`.

## Состав

- **track** — AI-ассистент трекера стартап-команд: фокус сессий, методологии (TOC/TRIZ), декомпозиция це
- **track-prep** — Подготовка к сессии трекинга стартап-команды
- **track-orch** — Оркестратор трекинга стартап-команд
- **track-debrief** — Разбор после сессии трекинга стартап-команды
- **l-t-challenge** — Self-challenge трекера — извлечь пары 'вопрос команды → твоя позиция' из сессии и методиче
- **l-t-svetofor** — Светофор + спидометр — быстрый чек прогресса команды за трекинговую сессию
- **l-t-jtbd** — JTBD-анализ интервью по методологии 'Спроси маму' (Mom Test)
- **l-t-jtbd-fit** — JTBD Fit Check — проверка product-market fit спеки по Jobs-to-be-Done (Christensen, Ulwick
- **thesis-extraction** — Извлечение ВСЕХ тезисов из транскрипта: кластеризация по темам, граф связей (дерево веток 
- **l-w-granola** — Универсальный анализ встречи из Granola-транскрипта
- **flyinglogic** — Генератор диаграмм Flying Logic (нативный формат .xlogic через JSON+генератор) и деревьев 
- **l-a-ikr** — ИКР — Идеальный Конечный Результат (ТРИЗ)
- **l-a-rat** — Risk Assumption Test — анализ рисков продуктовой идеи по Strategyzer
