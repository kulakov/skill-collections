# RAT (Risk Assumption Test) -- Шаблон

## Принцип

Начинать с гипотезы, опровержение которой сильнее всего повлияет на бизнес.

**Формула:** Стоимость риска = Вероятность ошибки x Влияние на цель

> **Режим по умолчанию: Трекшн-карта 2.0 (Distribution First).**
> Классический порядок — ниже, для deep tech / enterprise / regulated markets.
> Источник истины: `11-TRACKER/TRACTION_MAP_v2.md`
