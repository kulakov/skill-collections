# ИКР - Идеальный Конечный Результат

Примени методологию ТРИЗ для анализа проблемы.

## Шаг 1: Формулировка задачи
Опиши исходную проблему максимально конкретно.

## Шаг 2: Определение ИКР
Сформулируй Идеальный Конечный Результат:
"Система САМА [выполняет нужную функцию], БЕЗ [затрат, сложностей, побочных эффектов]"

## Шаг 3: Выявление противоречий

**Техническое противоречие:**
Если улучшить [параметр A], то ухудшится [параметр B]

**Физическое противоречие:**
[Элемент системы] должен быть [свойство X] чтобы [польза], и должен быть [НЕ-свойство X] чтобы [другая польза]

## Шаг 4: Ресурсы системы
Какие ресурсы уже есть и не используются:
- Вещественные (материалы, отходы, среда)
- Энергетические (поля, движение, тепло)
- Пространственные (пустоты, поверхности)
- Временные (до, после, паузы)
- Информационные (данные, сигналы)
- Функциональные (другие элементы системы)

## Шаг 5: Приемы разрешения противоречий
- Разделение в пространстве
- Разделение во времени
- Разделение между целым и частями
- Разделение по условию
- Переход в другую систему

## Шаг 6: Генерация решений
На основе анализа предложи 3-5 концептуальных решений, приближающих к ИКР.

## Шаг 7: Оценка решений
Для каждого решения:
- Насколько приближает к ИКР (%)
- Какие ресурсы требует
- Какие новые проблемы создает

Проблема для анализа:
$ARGUMENTS
