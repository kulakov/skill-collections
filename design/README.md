# Дизайн

Скиллы для дизайна и UX: дизайн-системы, UX-аудит, лендинги, Figma, генерация изображений, слайды, стили.

**24 скиллов.** Положи папки/файлы в `~/.claude/skills/` и `~/.claude/commands/`.

## Состав

- **add-style** — Добавление нового стиля в библиотеку стилей изображений
- **design-oracle** — Извлекает дизайн-систему живого сайта по URL через браузер (Playwright): цвета, типографик
- **frontend-design** — Выразительные production-grade фронтенд-интерфейсы высокого качества
- **l-d-decompose** — Разрезатор референсов на приёмы — декомпозиция лендинга/сайта на 7 слоёв дизайн-решений
- **l-d-design-system** — Извлечение портативной дизайн-системы из декомпозиции референса
- **l-d-ds-codify** — Кодификация дизайн-системы проекта: аудит CSS, создание токенов, LLM-readable спеки, скрип
- **l-d-enhance** — Апскейл изображений через AuraSR v2 (4x, GAN, локально на MPS)
- **l-d-figma** — Figma <-> Code раундтрип
- **l-d-img-edit** — Редактор картинок с контролем композиции
- **l-d-landing** — Методика структуры и копирайтинга лендинга (StoryBrand, PAS, AIDA)
- **l-d-slides** — Генератор HTML-презентаций из markdown
- **l-d-vibe-landing** — Вайб-кодинг лендингов по референсу
- **l-g-portrait** — Промпт для генерации портретов в Midjourney/DALL-E
- **l-s-slide-check** — Проверка композиции HTML-презентации
- **l-s-slide-fit** — Подгонка контента под слайды до генерации HTML
- **map-maker** — Генератор карт вымышленного мира. World-as-data: YAML источник истины + nano-banana растро
- **nanobanana** — Генерация изображений через Gemini API
- **page-cro** — Оптимизация конверсии страниц (CRO)
- **prd-review** — Ревью PRD документа по чеклисту качества
- **site-architecture** — Иерархия страниц сайта, навигация, URL-структура, внутренняя перелинковка
- **style-profile** — ОСНОВНОЙ скилл для клонирования визуального стиля. Извлекает JSON brand-style-profile из 1
- **ui-ux-pro-max** — Дизайн-система и UX-справочник для веба и мобайла
- **ux-audit-rethink** — Комплексный UX-аудит по 7 факторам IxDF, 5 характеристикам юзабилити, 5 измерениям взаимод
- **ux-scenario-redesign** — UX+UI редизайн под целевой сценарий
